# 🚀 Introduction to Git & GitHub

Git and GitHub are essential tools used by developers and software companies to manage code, collaborate with teams, and build projects efficiently.

Whether you are a beginner or preparing for industry jobs, understanding these tools is highly valuable.

---

## 📌 What is Git?

**Git** is a **Distributed Version Control System (VCS)** created to track changes in source code over time.

It helps developers manage project history, work safely, and collaborate without losing code.

## 🎯 Why Git is Used?

- Save project history
- Undo mistakes easily
- Track who changed code
- Work with team members
- Manage multiple versions
- Create separate branches for features
- Merge completed work safely

## ☁️ What is GitHub?

GitHub is a cloud platform used to host Git repositories online.

It provides a place where developers can store code, share projects, review code, and collaborate from anywhere.

## 🚀 Why Companies Use GitHub?

- Online repository hosting
- Team collaboration
- Pull Requests
- Code Reviews
- Backup of code
- CI/CD integrations

## 📘 Important Terms

- Repository = Project folder managed by Git
- Commit = Saved snapshot of changes
- Branch = Separate line of development
- Merge = Combine code changes
- Pull Request = Request to review and merge code

## ⭐ Final Note

Git helps manage your code.  
GitHub helps manage your team.
